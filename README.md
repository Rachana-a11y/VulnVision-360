# 🔐 Linux Security Hardening & Compliance Project

## 📖 Overview
This project focuses on implementing Linux system security using industry-standard tools such as OpenSCAP and Ansible. The objective is to perform vulnerability scanning, apply security hardening, and validate improvements through compliance reports.

---

## 🧰 Tools & Technologies Used
- Ubuntu 24.04
- Kali Linux
- OpenSCAP
- SCAP Security Guide (SSG)
- Ansible
- VirtualBox
- SSH

---

## 📅 Week 1 – Environment Setup

### 🎯 Objective
Set up lab environment for security testing.

### ⚙️ Tasks Performed
- Installed Ubuntu and Kali Linux in VirtualBox
- Configured network settings
- Verified connectivity using ping
